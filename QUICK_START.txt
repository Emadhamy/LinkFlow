🎯 الطرق المتاحة:

1️⃣  PWA Builder (الأسهل - 5 دقائق)
   ↳ اذهب إلى: https://www.pwabuilder.com/
   ↳ أدخل رابطك، واختر Android
   ↳ حمّل APK تلقائياً

2️⃣  Cordova CLI (محلي - 30 دقيقة)
   ↳ شغّل: ./build-apk.sh build
   ↳ اتبع التعليمات على الشاشة

3️⃣  Docker (موثوق - 45 دقيقة)
   ↳ شغّل: docker build -t linkflow .
   ↳ ثم: docker run --rm linkflow-builder

4️⃣  GitHub Actions (CI/CD)
   ↳ استخدم: .github/workflows/
   ↳ البناء التلقائي عند كل push

⚡ الخطوات السريعة (Cordova):

  # 1. تثبيت المتطلبات
  npm install -g cordova
  sdk install java 17.0.17-amzn
  
  # 2. تشغيل السكريبت
  chmod +x build-apk.sh
  ./build-apk.sh build
  
  # 3. البحث عن APK
  # الملف في: linkflow-cordova/platforms/android/app/build/outputs/apk/

❓ مشاكل شائعة:

  ❌ ANDROID_HOME not found
     ↳ export ANDROID_HOME=~/Android/Sdk

  ❌ Java version error
     ↳ sdk use java 17.0.17-amzn

  ❌ gradlew permission denied
     ↳ chmod +x linkflow-cordova/platforms/android/gradlew
