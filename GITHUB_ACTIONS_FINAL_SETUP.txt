الخطوة 1: اعمل commit و push
  $ git add .
  $ git commit -m "🚀 Setup GitHub Actions"
  $ git push origin main

الخطوة 2: اذهب إلى GitHub
  → Repository → Actions tab
  → ستجد workflows جاهزة

الخطوة 3: انتظر البناء
  ⏱️  15-20 دقيقة (أول مرة)

الخطوة 4: حمّل APK
  → Artifacts section
  → LinkFlow-Debug أو LinkFlow-Release

أثناء التطوير (Development):
  $ git push origin main
  ✅ يُبنى APK Debug تلقائياً

عند إنشاء إصدار (Release):
  $ git tag v1.0.0
  $ git push origin v1.0.0
  ✅ يُبنى Release APK ويُنشر على GitHub

بناء يدوي (Manual):
  GitHub → Actions → Build & Sign APK
  → Run workflow → اختر debug/release
  ✅ بناء حسب الطلب
